# Штурман - твой навигатор в мире автоспорта

Сайт-агрегатор автоспортивных событий России: **ралли, дрифт, кольцо и мото** в одном месте.
Календарь гонок, страницы событий с гидом для новичка, база знаний и промо мини-приложения.

> MVP-демо. Данные о событиях - ориентировочные (сезон 2026), хранятся в `assets/data/` и легко редактируются.

## Страницы

| Файл | Назначение |
|------|-----------|
| `index.html` | Главная: hero, афиша, дисциплины, промо приложения, блог |
| `calendar.html` | Календарь с фасетными фильтрами (дисциплина, регион, цена, месяц, поиск) |
| `event.html?id=…` | Страница гонки: дата, трасса, тип, билеты, «что это», «чем заняться зрителю», факт |
| `blog.html` | База знаний фаната (статьи) |
| `article.html?slug=…` | Страница статьи |
| `app.html` | Промо мини-приложения ВКонтакте |

## Технологии

- Статичный сайт: **HTML + Tailwind CSS + ванильный JS**, без сборки.
- Шрифты: Russo One (заголовки), Manrope (интерфейс), Inter (текст).
- Данные - в JS-объектах (`assets/data/events.js`, `articles.js`), работают офлайн.
- Адаптив, доступность, бережная анимация (учитывает `prefers-reduced-motion`).

## Запуск локально

Просто откройте `index.html`, либо поднимите статический сервер:

```bash
python -m http.server 8000
# затем http://localhost:8000
```

## Структура

```
├── index.html, calendar.html, event.html, blog.html, article.html, app.html
└── assets/
    ├── css/app.css        - дизайн-система (токены, компоненты, анимации)
    ├── js/                - ui.js (общий слой) + скрипты страниц
    └── data/              - events.js, articles.js (контент)
```
